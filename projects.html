Proyek Saya

Sistem Kontrol PID untuk Heat Exchanger

Januari 2022 PLC, PID, SCADA

Mendesain dan mengimplementasikan sistem kontrol PID untuk mengatur suhu pada heat exchanger industri menggunakan PLC Siemens S7-1200 dan antarmuka HMI. Sistem mampu mengurangi overshoot sebesar 40% dan meningkatkan stabilitas proses.

Fitur Utama:

  • Pengaturan parameter PID secara real-time
  • Antarmuka SCADA untuk monitoring
  • Alarm dan sistem proteksi
  • Data logging untuk analisis

Teknologi Digunakan:

Siemens TIA Portal WinCC SCADA PID Compact Modbus TCP

Sistem Monitoring Energi Berbasis IoT

Agustus 2021 ESP32, MQTT, Cloud

Mengembangkan sistem monitoring konsumsi energi real-time untuk pabrik menggunakan ESP32 dengan sensor arus/PZEM-004T. Data dikirim ke cloud melalui MQTT dan divisualisasikan dalam dashboard berbasis web dengan analisis tren harian/mingguan.

Fitur Utama:

  • Pengukuran daya, arus, tegangan real-time
  • Notifikasi melebihi threshold
  • Antarmuka web responsif
  • Integrasi dengan sistem ERP perusahaan

Teknologi Digunakan:

ESP32 Node-RED InfluxDB Grafana Mosquitto MQTT

Custom Industrial Controller Board

Maret 2020 PCB, STM32, Embedded

Mendesain dan memproduksi custom controller board berbasis STM32 untuk aplikasi khusus di industri makanan. Board ini menggantikan solusi komersial yang mahal dengan fungsionalitas yang disesuaikan dengan kebutuhan proses khusus klien.

Fitur Utama:

  • 4 channel input analog 4-20mA
  • 8 channel output relay
  • Komunikasi RS485/Modbus
  • Proteksi surge dan isolasi

Teknologi Digunakan:

STM32F103 Altium Designer Embedded C Modbus RTU
